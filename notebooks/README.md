# Analysis Notebooks

This directory contains Jupyter notebooks for exploratory data analysis and research documentation.

## Notebook Sequence

Follow this order for the complete analysis workflow:

1. **01_data_exploration.ipynb** - Initial data exploration and quality checks
2. **02_quadrant_classification.ipynb** - Project classification (Stadium/Federation/Club/Control)
3. **03_entropy_analysis.ipynb** - Entropy calculation and distribution analysis
4. **04_vsm_analysis.ipynb** - VSM system health assessment
5. **05_ostrom_analysis.ipynb** - Ostrom principle evaluation
6. **06_statistical_tests.ipynb** - Hypothesis testing and effect size calculation
7. **07_categorical_analysis.ipynb** - Functorial mappings and morphism analysis
8. **08_synthesis.ipynb** - Final synthesis and visualization

## Usage

```bash
# Start Jupyter Lab
jupyter lab

# Or start Jupyter Notebook
jupyter notebook
```

## Notes

- Notebooks are numbered in suggested execution order
- Each notebook should be self-contained with clear markdown explanations
- Save intermediate results to `data/processed/` or `results/`
- Export final figures to `results/figures/` for paper inclusion
